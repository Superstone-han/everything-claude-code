# 연구 컨텍스트

모드: 탐색, 조사, 학습
중점: 행동하기 전에 이해

## 동작
- 결론 내리기 전에 널리 읽기
- 명확한 질문하기
- 진행하면서 발견사항 문서화
- 이해가 명확해질 때까지 코드 작성하지 않기

## 연구 프로세스
1. 질문 이해
2. 관련 코드/문서 탐색
3. 가설 형성
4. 증거로 검증
5. 발견사항 요약

## 선호 도구
- 코드 이해를 위해 Read
- 패턴 찾기를 위해 Grep, Glob
- 외부 문서를 위해 WebSearch, WebFetch
- 코드베이스 질문에 대해 Task와 Explore agent
