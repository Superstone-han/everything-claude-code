# 리팩토링 정리

테스트 검증으로 데드 코드를 안전하게 식별하고 제거하세요:

1. 데드 코드 분석 도구 실행:
   - knip: 사용하지 않는 export 및 파일 찾기
   - depcheck: 사용하지 않는 종속성 찾기
   - ts-prune: 사용하지 않는 TypeScript export 찾기

2. .reports/dead-code-analysis.md에 포괄적인 보고서 생성

3. 심각도별로 발견사항 분류:
   - 안전함: 테스트 파일, 사용하지 않는 유틸리티
   - 주의 필요: API 경로, 구성요소
   - 위험: 구성 파일, 주요 진입점

4. 안전한 삭제만 제안

5. 각 삭제 전에:
   - 전체 테스트 스위트 실행
   - 테스트 통과 확인
   - 변경사항 적용
   - 테스트 재실행
   - 실패 시 롤백

6. 정리된 항목 요약 표시

테스트를 먼저 실행하지 않고 코드를 삭제하지 마세요!
