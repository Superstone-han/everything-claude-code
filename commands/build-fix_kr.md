# 빌드 및 수정

TypeScript 및 빌드 오류를 점진적으로 수정하세요:

1. 빌드 실행: npm run build 또는 pnpm build

2. 오류 출력 파싱:
   - 파일별로 그룹화
   - 심각도별로 정렬

3. 각 오류에 대해:
   - 오류 컨텍스트 표시 (앞뒤 5줄)
   - 문제점 설명
   - 수정 제안
   - 수정 적용
   - 빌드 재실행
   - 오류 해결 확인

4. 다음 경우 중지:
   - 수정으로 새 오류가 발생할 때
   - 동일한 오류가 3번 시도 후에도 지속될 때
   - 사용자가 일시 중지를 요청할 때

5. 요약 표시:
   - 수정된 오류
   - 남은 오류
   - 새로 도입된 오류

안전을 위해 한 번에 하나의 오류를 수정하세요!
